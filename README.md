# Valutatrade Hub 

Консольный валютный кошелек на Python для управления мультивалютным портфелем.

## Функции

-  Регистрация и аутентификация пользователей
-  Управление мультивалютным портфелем
-  Просмотр курсов валют в реальном времени
-  Покупка и продажа валют
-  Отслеживание баланса в разных валютах
-  Безопасное хранение данных

##  Установка

```bash
# Установка зависимостей
make install

# Запуск приложения
make project
